Closure self-audit: diff `raw.md` against `final.md` and correlate with `review/raw.review.md`.

Workspace: `$ARGUMENTS`

### Preflight

Confirm all three files exist; if any is missing, stop and report:

- `<workspace>/raw.md`
- `<workspace>/final.md`
- `<workspace>/review/raw.review.md`

(`final.md` is typically generated by `scripts/apply_review.py`. If absent, prompt the user to run `/ocr` or apply_review manually.)

### Execute

```
python3 skills/diff-review/scripts/md_diff.py \
  --raw <workspace>/raw.md \
  --final <workspace>/final.md \
  --review <workspace>/review/raw.review.md \
  --out <workspace>/previews/diff-review.html \
  --summary <workspace>/review/diff-summary.md
```

### Report

- HTML absolute path: `<workspace>/previews/diff-review.html`
- Summary absolute path: `<workspace>/review/diff-summary.md`
- Counts across the four states: `accepted` / `missed` / `outside-checklist` / `unanchored`
- If any `missed` or `outside-checklist` items appear, list them individually — this is the core output of the self-audit.
