## 1. OCR 常见字形混淆

现代简体 OCR 虽成熟，5 号字 / 扫描模糊 / 铅印磨损时仍会错。重点关注：

### 一字之差的经典

| OCR 易出 | 正字 | 线索 |
|---------|------|------|
| 曰 / 日 | 按语义 | 「子曰」必曰 |
| 己 / 已 / 巳 | 按语义 | 已经 / 自己 / 辰巳 |
| 未 / 末 | 按语义 | 未来 / 末年 |
| 戊 / 戌 / 戍 / 成 | 按语义 | 戊戌 / 戍守 / 成败 |
| 土 / 士 | 按语义 | 土地 / 士兵 |
| 兔 / 免 | 按语义 | 兔子 / 免除 |
| 木 / 本 | 按语义 | 树木 / 根本 |
| 人 / 入 | 按语义 | 人民 / 进入 |
| 干 / 千 | 按语义 | 干部 / 一千 |
| 大 / 太 / 夭 / 夫 | 按语义 | OCR 对中间那一点敏感 |
| 币 / 市 | 按语义 | 货币 / 城市 |

## 2. 标点符号规范（现代）

| 元素 | 规范 |
|------|------|
| 句号 | `。`（不是 `.`） |
| 逗号 | `，` |
| 顿号 | `、` |
| 分号 | `；` |
| 冒号 | `：` |
| 双引号 | `""`（对齐打开关闭） |
| 单引号 | `''` |
| 书名号 | `《》`嵌 `〈〉` |
| 省略号 | `……`（六个点占两格，不是 `...` 或 `…`） |
| 破折号 | `——`（双破折号占两格，不是 `-` 或 `—`） |
| 连接号 | `-`（数字范围）/ `——`（可替代） |
| 间隔号 | `·`（外国人名间隔 / 月日分隔） |
| 英文句末 | 英文句末用英文 `.`，中文句末用中文 `。`；不混用 |

### OCR 常错标点

| OCR 输出 | 应作 | 场景 |
|---------|------|------|
| `.` | `。` | 中文句末 |
| `...` | `……` | 中文省略 |
| `-` 连续多个 | `——` | 破折号 |
| `'` `"` 直引号 | `"` `'` 弯引号 | 中文文本 |
| `，。` 紧跟 | `。` 或 `，` 之一 | 不要重复 |
| `"` 不配对 | `""` 配对 | 开头用 `"`，结尾用 `"` |

## 3. 引文格式（GB/T 7714）

国家标准 GB/T 7714 是学术期刊通用格式。JN 投稿时大多刊物遵循。常见格式：

### 期刊论文

```
[1] 张三. 论某某问题[J]. 历史研究, 2020(3): 45-60.
```

### 图书

```
[2] 李四. 中国近代史[M]. 北京: 中华书局, 2018: 120.
```

### 译著

```
[3] 托克维尔. 旧制度与大革命[M]. 冯棠, 译. 北京: 商务印书馆, 1992.
```

### 会议论文

```
[4] 王五. 试论某某[C]//中国史学会. 第三届年会论文集. 北京: 中国社会科学出版社, 2019: 200-215.
```

### 档案 / 未刊文献

```
[5] 中国第二历史档案馆. 档案号[B]. 全宗号-目录号-案卷号.
```

**校对重点**：
- `[J]` `[M]` `[C]` 等文献类型标识符是否正确
- 编者 / 译者信息是否齐全
- 页码范围格式是否统一（`45-60` 不是 `45～60`）
- 期刊名是否用 `《》`（JN 领域常用，非 `""` 非 `< >`）
- 出版社 + 出版年是否齐备

## 4. 引号嵌套层级

中文论文引号嵌套规则：

```
外层双引号 "..."
 └ 内层单引号 '...'
```

举例：
```
他说："我看过《史记》里'究天人之际'的说法，印象很深。"
```

**常见错**：
- 外单内双：`'他说："好"'` ← 错，应外双内单
- 英文引号 `" "` 混用：中文正文用中文引号，只有英文内容才用英文引号

## 5. 破折号、连接号、省略号

这三个是 OCR 最常错的：

| 符号 | 宽度 | 用途 |
|------|------|------|
| `—` 单破折号 | 1 格 | 连字符（较少用） |
| `——` 双破折号 | 2 格 | 语义转折、解释 |
| `-` 半角连字号 | 0.5 格 | 英文、数字范围 |
| `～` 波浪号 | 1 格 | 数字范围（非正式） |
| `…` 单省略号 | 1 格 | 较少用 |
| `……` 双省略号 | 2 格 | 中文省略标准 |
| `...` 三点 | 1.5 格 | **不用**于中文正文 |

## 6. 年份与日期格式

中文学术写作年份格式：

| 正确 | 错误 |
|------|------|
| 1937 年 7 月 7 日 | 1937.7.7 |
| 20 世纪 80 年代 | 80 年代 / 1980s |
| 民国二十六年（1937） | 民国 26 年 |
| 公元前 221 年 | BC 221 / 前 221 |

数字：
- 时间、年份、公历日期、统计数字、计量单位前 → **阿拉伯数字**
- 概数（几十、成百上千、大约二三十）→ **汉字**
- 朝代纪年 → **汉字**（「乾隆四十二年」不是「乾隆 42 年」）

## 7. 专名规范

### 国号 / 政权名
- 「中华人民共和国」（全）/「中国」（简）—— 分场合
- 「中华民国」—— 指 1912-1949 政权，研究写作保留
- 「清朝」「清代」「清」—— 通用
- 「北洋政府」「南京国民政府」—— 历史专名，准确

### 地名
- 历史地名保留当时称谓（「长安」不改「西安」）
- 现代地名用官方名（「呼和浩特」不用「歸綏」除非引原文）
- 自治区全称：内蒙古 / 新疆 / 广西 / 宁夏 / 西藏（后接"自治区"）

### 机构名
- 高校：首次出现用全称，之后可简称
- 研究机构：「中国社会科学院」（全）/ 「社科院」（简）
- 档案馆：「中国第一历史档案馆」「中国第二历史档案馆」

## 8. 学术写作常见错

| 错 | 对 | 说明 |
|----|----|------|
| 的、地、得 混用 | 按语法 | 「我的书」「快快地跑」「跑得快」 |
| 做、作 混用 | 按语义 | 「做饭」「作品」 |
| 其它 / 其他 | 其他 | 「其它」不规范 |
| 做为 / 作为 | 作为 | |
| 溶化 / 融化 / 熔化 | 按语义 | 盐溶于水 / 雪融化 / 金属熔化 |
| 截止 / 截至 | 按语义 | 报名截止 / 截至昨日 |
| 国藉 / 国籍 | 国籍 | OCR 常错 |

## 9. OCR 常见格式错

- **页眉页脚残留**：OCR 没去干净会把"第 X 期""总第 X 期"当成正文——A 类删
- **表格识别乱**：复杂表格 OCR 可能把表头和数据混一行——重建表格
- **脚注残留**：脚注标记 `[1]` 被当成正文——重建为脚注块
- **数字上下标**：温度 `25℃`、化学式 `CO₂`、指数 `10²`——OCR 常丢失上下标

## 10. agent 使用此 reference 的方式

- 字形近似组合（第 1 节）+ 语义矛盾 → A 类 OCR 错
- 标点格式不规范（第 2 节）→ B 类规范问题
- 引文格式不规范（第 3 节）→ B 类规范问题
- 专名拿不准（第 7 节）→ C 类查证
- 句段语义跳跃、格式异常（第 9 节）→ C 类告诉 JN 可能是 OCR 没识别干净
