## 2. 旧式标点（保留，不要现代化）

民国普及现代标点 (1920s 左右)，用法与今略有差别：

| 符号 | 民国用法 | 现代 |
|------|---------|------|
| `．` | 句号（下面一个点） | `。` |
| `、、` | 叠用表强化并列 | `、` 单用 |
| `——`（单破折号） | 相当于现代破折号 | `——`（双破折号占两格） |
| `『』` | 引号（双书名号常见） | `""` 或 `「」` |
| 「」 | 引号 | `""` |
| `[]` / `〈〉` | 编者注 / 补字 | 现代仍用但减少 |
| `／` | 分隔、选择 | 同 |
| `、` | 顿号 | 同 |

**OCR 常误识**：`．`（民国句号）→ `.`（英文句点）→ 影响语感。看全文用字判断要不要改。如果整篇都用 `．`，保留；如果 OCR 某处出现英文 `.` 可能是误识。

## 5. 民国期刊版式特征

### 竖排左翻 vs 横排右翻
民国早期报刊多**竖排右起左翻**（古籍式），1930 年代后部分改**横排左起右翻**。OCR 如果方向判断错，整页文字倒序。

### 多栏分栏
报纸杂志常三栏、四栏、六栏排版。OCR 常按**从左到右逐行扫**而非**按栏逐栏读**，导致语义跳断。如果文本出现明显话题跳跃，检查是否分栏。

### 广告穿插正文
民国期刊常在正文间插广告块。OCR 会把广告和正文揉一起。读到突兀的商品 / 药品 / 补品推荐，是广告残留，建议删除。

### 注音字母 ㄅㄆㄇㄈ
民国识字教材常用。OCR 不识别 → 乱码 / 方框。见到乱码 + 教材 / 字词表上下文 → 标 C 类"疑为注音字母，需原件核对"。

## 7. 民国引文格式

民国学术文章的参考文献格式不统一：

- 早期：直接"某某曰"开头引用，无脚注
- 中期：章节后附说明，或用夹注 `（見某書）`
- 晚期：部分采用西式脚注，但格式混乱

**校对建议**：保留原貌，不要强行套 GB/T 7714。如用户要投现代期刊，by mp-format / to-docx 阶段再规范化。

- 见到字形差异先查第 1 节（异体），**不要标错**
